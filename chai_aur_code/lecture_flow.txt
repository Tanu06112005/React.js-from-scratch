-----------------------------------------------------------------------------------------------------------------------------------------------------------
                                TWO WAY BINDING
    
    ek esi approach jaha pr hm react ko hi kh deke h ki vo interact kre website se rather than the user itself interacting to the website
     Two-way binding ka matlab hai:

    Data UI (input) se State me jaye aur State se UI me wapas aaye.

    Yaani data dono directions me flow karta hai.

            User Input    
                │
                ▼
            React State
                ▲
                │
            Input Value


            react state('')--> userInput---->react State("user input save ho rha h")--> then input value


        *p-------Two-Way Binding in React---------*

Definition:

Two-way binding is a process where the UI (input field) and the state stay synchronized. Any change in the input updates the state, and any change in the state automatically updates the input.

In simple words:

Input → State (when the user types)
State → Input (when the state changes)

That's why it is called two-way binding.
