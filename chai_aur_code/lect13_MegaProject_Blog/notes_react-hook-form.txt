# React Hook Form (RHF)

## Why do we need it?

Normal React me har input ke liye alag `useState` aur `onChange` likhna padta hai.

```jsx
const [email, setEmail] = useState("");
const [password, setPassword] = useState("");
```

Agar 10 inputs hain to 10 states aur 10 `onChange` handlers.

👉 **React Hook Form is problem ko solve karta hai.**

---

## What is React Hook Form?

> **React Hook Form ek lightweight library hai jo form data, validation aur errors ko manage karti hai bina har input ke liye alag `useState` banaye.**

---

# Main Hooks

### 1. `useForm()`

Form ko initialize karta hai.

```jsx
const {
  register,
  handleSubmit,
  watch,
  getValues,
  setValue,
  reset,
  formState: { errors }
} = useForm();
```

---

### 2. `register()`

Input ko React Hook Form ke saath connect karta hai.

```jsx
<input {...register("email")} />
```

* Input ko register karta hai.
* Value aur validation automatically track hoti hai.

**`...register()` kyu?**

`register()` ek object return karta hai:

```jsx
{
  name,
  onChange,
  onBlur,
  ref
}
```

Spread (`...`) us object ki saari properties input ko pass kar deta hai.

---

### 3. `handleSubmit()`

Form submit hone par validation karta hai.

```jsx
<form onSubmit={handleSubmit(onSubmit)}>
```

✔ Valid → `onSubmit(data)` call hoga.

❌ Invalid → `errors` fill ho jayega.

---

### 4. `errors`

Validation errors store karta hai.

```jsx
errors.email
```

---

### 5. Validation

```jsx
{...register("email", {
  required: "Email is required"
})}
```

Supports:

* `required`
* `minLength`
* `maxLength`
* `pattern`
* `validate`

---

### 6. `watch()`

Input ki **live value** observe karta hai.

```jsx
const email = watch("email");
```

---

### 7. `getValues()`

Current value ek baar return karta hai.

```jsx
getValues("email")
```

---

### 8. `setValue()`

Input ki value manually set karta hai.

```jsx
setValue("name", "Tanushika");
```

---

### 9. `reset()`

Form reset karta hai.

```jsx
reset();
```

---

# Flow

```text
User types
      ↓
register()
      ↓
RHF stores values
      ↓
Submit
      ↓
handleSubmit()
      ↓
Validation
      ↓
errors (if invalid)
      ↓
onSubmit(data) (if valid)
```

---

# Quick Notes

| Function         | Purpose            |
| ---------------- | ------------------ |
| `useForm()`      | Form initialize    |
| `register()`     | Input register     |
| `handleSubmit()` | Validate + Submit  |
| `errors`         | Validation errors  |
| `watch()`        | Live value         |
| `getValues()`    | Current value      |
| `setValue()`     | Set value manually |
| `reset()`        | Reset form         |

# ❌ Without React Hook Form

Agar Login Form hai jisme 3 fields hain:

* Name
* Email
* Password

To tumhe 3 states banani padengi.

```jsx
import { useState } from "react";

function Login() {

  const [name, setName] = useState("");
  const [email, setEmail] = useState("");
  const [password, setPassword] = useState("");

  const submit = () => {
    console.log({
      name,
      email,
      password
    });
  };

  return (
    <>
      <input
        placeholder="Name"
        value={name}
        onChange={(e) => setName(e.target.value)}
      />

      <input
        placeholder="Email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
      />

      <input
        type="password"
        placeholder="Password"
        value={password}
        onChange={(e) => setPassword(e.target.value)}
      />

      <button onClick={submit}>Submit</button>
    </>
  );
}
```

### Observe

Tumhe likhna pada

```jsx
const [name, setName] = useState("");
const [email, setEmail] = useState("");
const [password, setPassword] = useState("");
```

Aur har input ke liye

```jsx
value={name}
onChange={(e)=>setName(e.target.value)}
```

Same cheez baar-baar repeat ho rahi hai.

# ✅ With React Hook Form

```jsx
import { useForm } from "react-hook-form";

function Login() {

  const { register, handleSubmit } = useForm();

  const submit = (data) => {
    console.log(data);
  };

  return (
    <form onSubmit={handleSubmit(submit)}>

      <input
        placeholder="Name"
        {...register("name")}
      />

      <input
        placeholder="Email"
        {...register("email")}
      />

      <input
        type="password"
        placeholder="Password"
        {...register("password")}
      />

      <button type="submit">
        Submit
      </button>

    </form>
  );
}
```

---

## Ab dhyan do...

Humne kitni states banayi?

```jsx
const [name, setName]
const [email, setEmail]
const [password, setPassword]
```

👉 **Ek bhi nahi.**

Humne `value` bhi nahi diya.

Humne `onChange` bhi nahi likha.

Sirf

```jsx
register("name")
```

likh diya.

---

## Data kaha store hua?

Jab user type karta hai

```
Name -> Tanushika
Email -> abc@gmail.com
Password -> 123456
```

React Hook Form internally sab values ko store karta rehta hai.

Submit pe

```jsx
const submit = (data) => {
    console.log(data);
}
```

`data` automatically ban jata hai:

```js
{
   name: "Tanushika",
   email: "abc@gmail.com",
   password: "123456"
}
```

Tumhe manually object bhi nahi banana padta.

---

## Internally kya hota hai?

```
User types
      │
      ▼
<input {...register("email")} />
      │
      ▼
React Hook Form stores value internally
      │
      ▼
Submit
      │
      ▼
handleSubmit()
      │
      ▼
submit(data)
```

---

## 📌 Notes

**Without React Hook Form**

* Har input ke liye `useState`.
* Har input me `value` aur `onChange`.
* Form data manually collect karna padta hai.

**With React Hook Form**

* `useState` ki zarurat nahi.
* Sirf `register()` se input connect ho jata hai.
* Form submit hote hi saara data ek object (`data`) me automatically mil jata hai.
