### Ek chhoti si Java comparison (kyunki tum Java bhi padh rahe ho)

```java
Scanner sc = new Scanner(System.in);
```

* `Scanner` → Class
* `sc` → Object

```javascript
Client client = new Client();
```

* `Client` → Appwrite SDK ki class
* `client` → Us class ka object

Aur jaise Java me `Scanner` ke methods hote hain (`nextInt()`, `nextLine()`), waise hi `Client` ke methods hain (`setEndpoint()`, `setProject()`), aur `Account` ke methods hain (`create()`, `login()`, `logout()`).

Ye object-oriented programming ka hi use hai, bas Appwrite ne apni classes provide ki hain jo backend se communicate karna aasaan bana deti hain.
