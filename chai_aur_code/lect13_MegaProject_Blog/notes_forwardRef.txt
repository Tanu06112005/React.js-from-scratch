                        ## `forwardRef` (Concise Notes)
forwardRef - Interview Definition

forwardRef is a React API that allows a parent component to pass a ref to
 a child component, so the parent can directly access or manipulate the 
 child component's underlying DOM element.
 
### Why do we need it?

By default, **`ref` sirf HTML elements (`input`, `button`, `div`) par kaam karta hai.**
Agar `ref` kisi **function component** ko pass karte hain, to wo andar ke HTML element tak nahi pahuchta.

👉 Is problem ko solve karta hai **`forwardRef`**.

---

### What is `forwardRef`?

> **`forwardRef` parent component ke `ref` ko child component ke actual DOM element tak forward (pass) karta hai.**

---

### Without `forwardRef`

```jsx
const Input = () => <input />;

<Input ref={inputRef} />   ❌ Doesn't work
```

`ref` function component par hi ruk jata hai.

---

### With `forwardRef`

```jsx
const Input = React.forwardRef((props, ref) => {
  return <input ref={ref} {...props} />;
});
```

```jsx
const inputRef = useRef();

<Input ref={inputRef} />
```

Ab `inputRef.current` actual `<input>` ko point karega.

---

### Flow

```text
Without forwardRef

Parent
   │
   ▼
<Input />   ❌ ref stops here


With forwardRef

Parent
   │
   ▼
<Input />
   │
   ▼
<input /> ✅ ref reaches here
```

---

### Common Uses

* Focus an input (`inputRef.current.focus()`)
* Access DOM methods
* Build reusable Input, Button, Textarea components
* Third-party libraries

---

## 📝 Quick Notes

* `ref` by default **function components par work nahi karta.**
* `forwardRef` **ref ko child ke actual DOM element tak pahucha deta hai.**
* Mostly reusable components (`Input`, `Button`, `Textarea`) banate time use hota hai.

### ⭐ One-line Interview Definition

> **`forwardRef` is a React API that forwards a parent component's `ref` to a child component's DOM element, allowing the parent to directly access or control that element.**
